Esercizio completo di Backpropagation

📥 Input

  • Input: x = 30 (temperatura)
  • Target (y): un valore desiderato (es. 1)

🧠 Architettura

  • 1 input (x)
  • Hidden Layer 1: 2 neuroni (H1, H2)
  • Output Layer: 1 neurone (O)
  • Pesi: w1, w2 → H1; w3, w4 → H2; w5, w6 → Output
  • Bias: b1 (per H1), b2 (per H2), b3 (per Output)

⚙️ Calcoli Forward

  z1 = w1 * x + w2 * x + b1
  a1 = σ(z1) = 1 / (1 + e^(-z1))

  z2 = w3 * x + w4 * x + b2
  a2 = σ(z2) = 1 / (1 + e^(-z2))

  z3 = w5 * a1 + w6 * a2 + b3
  a3 = σ(z3) = output finale

🧮 Funzione di Costo

  L = (a3 - y)^2

🔁 Backpropagation

1. Output Layer Delta

  δo = (a3 - y) * σ'(z3)
     = (a3 - y) * a3 * (1 - a3)

2. Hidden Layer Delta

  δ1 = δo * w5 * σ'(z1) = δo * w5 * a1 * (1 - a1)
  δ2 = δo * w6 * σ'(z2) = δo * w6 * a2 * (1 - a2)

📦 Aggiornamento Pesi e Bias

Output Layer

  w5 = w5 - η * δo * a1
  w6 = w6 - η * δo * a2
  b3 = b3 - η * δo

Hidden Layer

  w1 = w1 - η * δ1 * x
  w2 = w2 - η * δ1 * x
  w3 = w3 - η * δ2 * x
  w4 = w4 - η * δ2 * x

  b1 = b1 - η * δ1
  b2 = b2 - η * δ2

📘 Nota

- σ(z) è la sigmoide: σ(z) = 1 / (1 + e-z)
- σ'(z) è la derivata della sigmoide: σ'(z) = σ(z)(1 - σ(z))
- η è il learning rate (es. 0.01)
