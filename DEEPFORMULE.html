Esempio Backpropagation - Solo Formule

1️⃣ Dati iniziali

  • Input: x = [x1, x2]
  • Hidden layer: 2 neuroni (h1, h2)
  • Output layer: 1 neurone
  • Target: y
  • Pesi e bias:
    • Hidden layer: w1, w2, w3, w4, b1, b2
    • Output layer: w5, w6, b3
  • Learning rate: η

2️⃣ Forward Propagation

Hidden layer:

z1 = x1*w1 + x2*w2 + b1

a1 = σ(z1)

z2 = x1*w3 + x2*w4 + b2

a2 = σ(z2)

Output layer:

z3 = a1*w5 + a2*w6 + b3

a3 = σ(z3)

3️⃣ Calcolo Delta Output

δo = (a3 - y) * σ'(z3) = (a3 - y) * a3 * (1 - a3)

4️⃣ Calcolo Delta Hidden Layer

δ1 = δo * w5 * σ'(z1) = δo * w5 * a1 * (1 - a1)

δ2 = δo * w6 * σ'(z2) = δo * w6 * a2 * (1 - a2)

5️⃣ Aggiornamento Pesi e Bias

Output layer:

w5 = w5 - η * δo * a1

w6 = w6 - η * δo * a2

b3 = b3 - η * δo

Hidden layer:

w1 = w1 - η * δ1 * x1

w2 = w2 - η * δ1 * x2

w3 = w3 - η * δ2 * x1

w4 = w4 - η * δ2 * x2

b1 = b1 - η * δ1

b2 = b2 - η * δ2
