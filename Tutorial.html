📐 Calcolo della Similarità del Coseno in Python

In questo tutorial calcoleremo quanto due immagini (sotto forma di vettori) sono simili tra loro usando la similarità del coseno.

📘 Formula matematica

similarità = (A · B) / (‖A‖ × ‖B‖)

Dove:
- A · B è il prodotto scalare
- ‖A‖ è la norma (lunghezza) del vettore A
- ‖B‖ è la norma (lunghezza) del vettore B

💻 Codice Python senza funzioni

# Due vettori (immagini appiattite in forma di lista)
A = [1, 2]
B = [3, 4]

# Calcolo del prodotto scalare A · B
prodotto_scalare = 0
for i in range(len(A)):
prodotto_scalare += A[i] * B[i]

# Calcolo della norma di A: √(a1² + a2² + ...)
somma_quadrati_A = 0
for i in range(len(A)):
somma_quadrati_A += A[i] * A[i]
norma_A = somma_quadrati_A ** 0.5

# Calcolo della norma di B
somma_quadrati_B = 0
for i in range(len(B)):
somma_quadrati_B += B[i] * B[i]
norma_B = somma_quadrati_B ** 0.5

# Calcolo della similarità del coseno
similarita_coseno = prodotto_scalare / (norma_A * norma_B)

# Output
print("Prodotto scalare (A · B):", prodotto_scalare)
print("Norma di A:", norma_A)
print("Norma di B:", norma_B)
print("Similarità del coseno:", similarita_coseno)

📈 Esempio numerico

Per i vettori A = [1, 2] e B = [3, 4]:
- Prodotto scalare = 1×3 + 2×4 = 11
- Norma A = √(1² + 2²) = √5 ≈ 2.236
- Norma B = √(3² + 4²) = √25 = 5
- Similarità = 11 / (2.236 × 5) ≈ 0.983

✅ Risultato

La similarità del coseno è circa 0.983: quindi i vettori sono molto simili!

🤖 Applicazione al riconoscimento immagini

Se trasformi due immagini in vettori (es. da 2×2 a [p1, p2, p3, p4]), puoi usare questa formula per dire quanto sono simili — anche se non sono pixel-per-pixel uguali!

📝 NOTE

  • Il confronto diretto con A == B funziona solo se i vettori sono identici.
  • La similarità del coseno funziona anche se i valori sono proporzionali (es. luce diversa).
  • Il valore sarà tra 0 (completamente diversi) e 1 (identici nella direzione).
