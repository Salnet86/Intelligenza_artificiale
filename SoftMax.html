Dati

Vettore di output (logits):

z = [1.2, 0.9, 2.5, 0.1, -0.5, 0, 1.1, 0.3, -1.0, 0.7]

Passo 1: Calcolare l’esponenziale di ogni numero

Per ogni numero in z, calcola e^(numero) (cioè "e alla potenza del numero"):

  • e^(1.2) = 3.32
  • e^(0.9) = 2.46
  • e^(2.5) = 12.18
  • e^(0.1) = 1.11
  • e^(-0.5) = 0.61
  • e^(0) = 1
  • e^(1.1) = 3.00
  • e^(0.3) = 1.35
  • e^(-1.0) = 0.37
  • e^(0.7) = 2.01

Passo 2: Sommare tutti questi valori

Somma tutti gli esponenziali:

S = 3.32 + 2.46 + 12.18 + 1.11 + 0.61 + 1 + 3 + 1.35 + 0.37 + 2.01 = 27.41

Passo 3: Dividere ogni esponenziale per la somma

Questo ti dà la probabilità di ogni classe:

  • Classe 1: 3.32 / 27.41 = 0.12 (12%)
  • Classe 2: 2.46 / 27.41 = 0.09 (9%)
  • Classe 3: 12.18 / 27.41 = 0.44 (44%)
  • Classe 4: 1.11 / 27.41 = 0.04 (4%)
  • Classe 5: 0.61 / 27.41 = 0.02 (2%)
  • Classe 6: 1 / 27.41 = 0.04 (4%)
  • Classe 7: 3 / 27.41 = 0.11 (11%)
  • Classe 8: 1.35 / 27.41 = 0.05 (5%)
  • Classe 9: 0.37 / 27.41 = 0.01 (1%)
  • Classe 10: 2.01 / 27.41 = 0.07 (7%)

Passo 4: Interpretazione

La classe con probabilità più alta è la Classe 3 con circa il 44% di probabilità.

Passo 5: Calcolo della perdita (loss) se la classe vera è la 3

La perdita si calcola come il logaritmo negativo della probabilità corretta:

loss = -log(0.44) ≈ 0.81
