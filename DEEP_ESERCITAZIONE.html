Esempio Backpropagation

1️⃣ Dati iniziali

  • Input: x = [0.9, 0.6]
  • Hidden layer: 2 neuroni (h1, h2)
  • Output layer: 1 neurone
  • Target: y = 1
  • Pesi e bias iniziali:
    • w1=0.88, w2=0.99, w3=0.6, w4=0.88
    • w5=0.9, w6=0.6
    • b1=0.1, b2=0.2, b3=0.05
  • Learning rate η = 0.1

2️⃣ Forward Propagation

Hidden layer:

z1 = x1*w1 + x2*w2 + b1 = 0.9*0.88 + 0.6*0.99 + 0.1 = 1.486

a1 = σ(z1) = 1 / (1 + e^-1.486) ≈ 0.815

z2 = x1*w3 + x2*w4 + b2 = 0.9*0.6 + 0.6*0.88 + 0.2 = 1.268

a2 = σ(z2) = 1 / (1 + e^-1.268) ≈ 0.780

Output layer:

z3 = a1*w5 + a2*w6 + b3 = 0.815*0.9 + 0.780*0.6 + 0.05 = 1.2515

a3 = σ(z3) = 1 / (1 + e^-1.2515) ≈ 0.777

3️⃣ Calcolo Delta Output

δo = (a3 - y) * a3*(1 - a3) = (0.777 - 1)*0.777*(1 - 0.777) ≈ -0.0386

4️⃣ Calcolo Delta Hidden Layer

δ1 = δo * w5 * a1*(1 - a1) = -0.0386 * 0.9 * 0.815*0.185 ≈ -0.00523

δ2 = δo * w6 * a2*(1 - a2) = -0.0386 * 0.6 * 0.780*0.22 ≈ -0.00397

5️⃣ Aggiornamento Pesi e Bias

Output layer:

w5 = w5 - η*δo*a1 = 0.9 - 0.1*(-0.0386*0.815) ≈ 0.90315

w6 = w6 - η*δo*a2 = 0.6 - 0.1*(-0.0386*0.780) ≈ 0.60301

b3 = b3 - η*δo = 0.05 - 0.1*(-0.0386) ≈ 0.05386

Hidden layer:

w1 = w1 - η*δ1*x1 = 0.88 - 0.1*(-0.00523*0.9) ≈ 0.88047

w2 = w2 - η*δ1*x2 = 0.99 - 0.1*(-0.00523*0.6) ≈ 0.99031

w3 = w3 - η*δ2*x1 = 0.6 - 0.1*(-0.00397*0.9) ≈ 0.60036

w4 = w4 - η*δ2*x2 = 0.88 - 0.1*(-0.00397*0.6) ≈ 0.88024

b1 = b1 - η*δ1 = 0.1 - 0.1*(-0.00523) ≈ 0.10052

b2 = b2 - η*δ2 = 0.2 - 0.1*(-0.00397) ≈ 0.20040
